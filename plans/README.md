## Findings considered and rejected

- **CORS AllowAnyHeader + AllowCredentials**: origens são whitelist explícita em `appsettings.json:34-36` — não é wildcard. Risco atual LOW. Registrar como revisão obrigatória ao adicionar novas origens.
- **X-Tenant-Slug header sem regex**: TenantBindingMiddleware (`Program.cs:157`) mitiga cross-tenant. Risco residual LOW; foi avaliado, não planejado.
- **ClockSkew = TimeSpan.Zero**: pode causar rejeições esporádicas em clock skew; risco LOW com NTP. Não planejado.
- **Docker fallback passwords**: afeta apenas dev local; `docker-compose.prod.yml` usa env vars. Não planejado.
- **BUG-14/15 (orphan customers em admin bookings)**: comportamento possivelmente intencional para bookings ad-hoc. Investigar com o time antes de planejar.
- **Auto-migrations em !IsProduction()**: mitigado pelo guard explícito; risco se environment for mal configurado. Baixa prioridade.
- **Scalar/Swagger em Caddyfile**: verificar se `Caddyfile` tem rota `/scalar/*` em produção — se sim, remover; se não, falso positivo.
- **RabbitMQ prefetchCount**: MassTransit configura via `cfg.UseMessageRetry` — comportamento padrão aceitável para o volume atual. Revisar em escala.
- **PERF-02 GetFinancialSummaryQuery filter após ToList**: verificar se `ToList()` precede o `Where` ou vice-versa — pode ser falso positivo do subagente. Investigar antes de planejar.
